An end and a start

May 6, 2021 in Random thoughts

Writing this blog after a very long time. I was trying to complete my FastAPI journey and somewhat succeeded too. However, the aim was to create tutorials on each and every topic. Tried to create some of them. Introduction to API, JSON and XML along with HTTP methods and status codes went great. Areeb Jamal and Prateek Jain helped a lot in reviewing them. Some of the tutorials are still pending as drafts. I remember my last conversation with Areeb Bhai about the blogs. He used to push me a lot to write but I failed. Not because I was procrastinating but because tutorials need time and a huge amount of effort in which my college curriculum played the role of villain. Nevertheless, I will still try my best to get them to publish probably during the summer vacations. But for now, the plans are different.

From now on I will write my daily/weekly updates in the form of blogs. Some of them will surely go long as I will continue my semester reports also. This idea came after I read Jason Braganza’s blogs on his journey of learning Django. His blogs are not tutorials but they are just recordings of his journey as he learns new things all over again.

So, from now on I will write everything I learn, everything I know about writing and everything else that I find interesting enough to share. Blogs from now on, won’t be much longer. They will be just updates of everything I could grab in a day.

But before starting this journey again let us look at what I have learned in FastAPI so far.

  1. What is REST API, HTTP methods and status codes?
  2. Why Python for API?
  3. Sync vs Async Python Frameworks
  4. How to think before starting a project?
  5. Python types and Pydantic
  6. What is versioning?
  7. Authentication/Authorization and middleware in FastAPI
  8. Swagger and auto-documentation
  9. Database with docker
  10. Naming Endpoints
  11. What is ORM and how it is different from pure SQL?
  12. Redis (in progress)
  13. Testing (in progress)
  14. Domain, Reverse Proxy, HTTPS and Auth 2.0 (in progress)

Tutorials on all these topics are very hard. However, I will write them one by one soon.
