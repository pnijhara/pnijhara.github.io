May 28, 2021 in AMU-OSS, Computer Science

No, this isn’t after a long time. Actually, yes, this blog is coming after a long time. I don’t know whether I should give justification or not, but yes, I got busy with a lot of other stuff. I got busy with my IEEE talk on Hugo, on which I have written 2 blogs. Along with the IEEE talk, I got occupied with the SSGSA bug again. This time it wasn’t a bug, it was an actual human error. We forgot to upload the javascript file for toggling the actions behind the buttons. Anyways, things are back to normal again.

In the past week, I managed to give some of the time to Operating Systems. Learnt a lot about Deadlocks.

What did I learn in the past week?

  1. Deadlock Avoidance
    • Using resource allocation graph (RAG)
    • Using Banker’s Algorithm
  2. Deadlock Detection
  3. Deadlock Recovery
  4. Deadlock Ignorance

Banker’s algorithm was the most interesting topic in the whole deadlock chapter. I was able to solve the trickiest problems with the banker’s algorithm. And, deadlock ignorance was the funniest topic that I read in the journey till now. I mean, after learning so much about the deadlocks, we came to the conclusion of ignoring deadlocks. Amazing!

Anyways, I am targetting to complete Disk Scheduling by the end of the week. Wish me luck!!
